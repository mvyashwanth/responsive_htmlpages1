# Responsive Authentication Web Application

## Project Description
This project is a responsive authentication web application built using **HTML, CSS, and Bootstrap**.  
It contains multiple authentication pages such as login, register, forgot password, and reset password.

The application follows modern UI practices with **clean layout, hover effects, box shadows, gradient backgrounds, and smooth transitions**.

---

## Features
- Responsive design for **Desktop, Laptop, Tablet, and Mobile**
- Hover effects for **buttons and links**
- **Box shadows** for cards
- **Gradient background styling**
- **Custom spacing**
- **Smooth transitions**
- Clean and well-structured code
- Professional UI design

---

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5

---

## Pages Included
1. **index.html** – Home Page  
2. **login.html** – Login Page  
3. **register.html** – Registration Page  
4. **forgot-password.html** – Forgot Password Page  
5. **reset-password.html** – Reset Password Page  

---

## Responsive Design
The application is tested for the following screen sizes:

- **Desktop:** 1920px and above  
- **Laptop:** 1366px – 1920px  
- **Tablet:** 768px – 1024px  
- **Mobile:** 320px – 767px
